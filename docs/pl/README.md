# Dokumentacja Biomass Explorer

Ten zestaw dokumentacji opisuje architekturę, działanie, konfigurację i utrzymanie aplikacji **Biomass Explorer**.

## Mapa dokumentacji

- `docs/pl/architektura.md` - architektura systemu, komponenty i przepływ runtime.
- `docs/pl/konfiguracja-i-uruchomienie.md` - instalacja lokalna, konfiguracja środowiska i podstawy wdrożenia.
- `docs/pl/referencja-api.md` - endpointy backendu, payloady i zasady dostępu.
- `docs/pl/baza-danych-i-bezpieczenstwo.md` - schemat PostgreSQL, zależności, reguły dostępu i bezpieczeństwo.
- `docs/pl/przewodnik-uzytkownika-i-admina.md` - zachowanie aplikacji dla gościa, użytkownika i administratora.

## Dla kogo

- **Developerzy**: szczegóły implementacyjne i rozszerzanie systemu.
- **DevOps/IT**: uruchomienie, konfiguracja środowisk, operacje DB.
- **Wsparcie i produkt**: zachowanie funkcjonalne i kontekst diagnostyczny.

## Źródło prawdy

Kod repozytorium pozostaje źródłem ostatecznym. Dokumentacja jest utrzymywana zgodnie z aktualnym zachowaniem gałęzi `feature/admin-panel-pg-i18n-next`.
